# CodeAlpha_Internship_HungmanGame

# Project 1: Text-Based Hangman Game
A classic, interactive, console-based Hangman game where players attempt to guess a randomly chosen secret word from a predefined list.

# Features
Random Word Selection: The game automatically picks a word from a predefined array of target words.

Input Validation: Handles incorrect inputs (such as numbers, multiple characters, or repeated guesses) without crashing the program.

Visual Progress Tracking: Displays correctly guessed letters in their exact positions and placeholders (_) for remaining hidden letters.

Turn Limit: Limits the user to exactly 6 incorrect guesses before triggers a game-over state.

# Key Concepts Utilized
Core control flow (while loops, if-elif-else conditionals).

Collection data types (lists, sets for tracking unique guesses).

The native Python random module.
